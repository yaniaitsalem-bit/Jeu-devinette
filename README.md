<h1> Jeu de Devinette 🎲 <h1/>

## Description
Mini-jeu en Python où le joueur doit deviner un nombre secret choisi aléatoirement par l'ordinateur. Le jeu propose plusieurs niveaux de difficulté et un système de scores.

## Fonctionnalités
- Menu principal interactif
- 3 niveaux de difficulté (Facile, Moyen, Difficile)
- Indices \"plus grand\" / \"plus petit\" à chaque tentative
- Compteur de tentatives
- Système de sauvegarde des meilleurs scores
- Possibilité de rejouer

## Comment lancer le jeu
```bash
python main.py
```

## Prérequis
- Python 3.6 ou supérieur
- Aucune bibliothèque externe requise (utilise uniquement `random` et `os`)
